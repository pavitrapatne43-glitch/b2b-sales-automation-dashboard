# B2B Sales Automation & Analytics Dashboard

## 📌 Project Overview
This project focuses on automating B2B lead management and building an interactive analytics dashboard.

## 🚀 Features
- Automated lead capture using Make.com
- Email notifications to clients and sales team
- Data storage using Google Sheets
- Interactive dashboard using Streamlit

## 📊 Dashboard KPIs
- Total Leads
- Converted Leads
- Conversion Rate
- Average Follow-Up Time

## 🛠 Tools Used
- Make.com (Automation)
- Google Forms & Sheets
- Python (Streamlit, Pandas)

## ▶️ How to Run
```bash
streamlit run app.py
